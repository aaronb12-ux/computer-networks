## MTU: Max Transmission Unit

Part of the datalink layer.

- Composed of:
  - IP header (header of the network layer)
  - TCP header
  - Data (called maximum segment size)

![MTU Structure](https://github.com/user-attachments/assets/ad0c2875-e789-4237-81f6-f990f4673f46)

## Full Duplex

- A → B and B → A (two different processes that communicate with each other)
- A sends out a sequence number to B and B ACKs it

**Important:** In TCP when a packet is ACKed, what you are acknowledging is the next element that you require.

**TCP uses byte-streamed sequencing**
